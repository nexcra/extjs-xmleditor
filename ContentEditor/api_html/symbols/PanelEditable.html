Class PanelEditable


Extends Ext.Panel.
PanelEditable
Defined in: PanelEditable.js.

Class Summary
Constructor Attributes Constructor Name and Description
 
PanelEditable()
Basic Ext component that is editable with REDO/UNDO support; extends Ext.Panel
Any content can be made editable by:
1) #initContent to place some content within the panel
2) #process to prepare nodes
3) #makeEditable to notify ContentEditor
Field Summary
Field Attributes Field Name and Description
 
marker
node attribute that ContentEditor looks for to make a node editable;
Can have more markers; however only the first one (if set) indicates 'editability'; the reminder is passed over to .updateElement - can be used to unique identification of editable nodes Note! only textnodes can be made editable
 
markerClassName
Depending on marker set bound to a node, ContentEditor sets nodes CSS names taken from the array
