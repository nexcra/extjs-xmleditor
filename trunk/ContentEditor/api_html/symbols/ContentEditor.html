ContentEditor()
Main component to provide editing fascility without the use of 'content editable'
Please refer to PanelEditable for details on how an Editor can be used.
Any content can be made editable by:
The basic idea is as follows:
1) issue ContentEditor.init() on a component that features scrolling 2) issue ContentEditor.makeEditable() on a dom node of the component
\> Requires editor.css
